# Com103-midterm-exam
WEEKLY EXPENSE TRACKER PROGRAM

WHAT THIS PROGRAM DOES:

Asks for the user's name and weekly budget
Lets the user input up to 4 expenses
Each expense is categorized (Food, Transportation, etc.)
Calculates total spending and remaining budget
Shows a summary report at the end
Warns if an expense is too high or if the user overspends

HOW TO RUN:

Run this program in a Python interpreter (IDLE, VS Code, or terminal)
Enter your name when prompted
Enter your weekly budget (numbers only)
Choose a category (1–5) for each expense or 0 to skip
Input description and amount for each expense
View your final budget summary at the end
